# Cascader 级联组件 beta

当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择。

# 基础用法

<l-cascader :source.sync="source"
            :selected.sync="selected"
            @update:selected="fn"
            popover-height="200px">
</l-cascader>

# 动态加载

当选中某一级时，动态加载该级下的选项

selected : null null null
<l-cascader :source.sync="source"
            :selected.sync="selected"
            @update:selected="fn"
            popover-height="200px"
            :load-data="loadData">
</l-cascader>

<script>
    methods(){
        loadData(item, updateSource) {
            let id = item.id;
            ajax(id).then(result => {
                updateSource(result);
            });
        },
    }
</script>
