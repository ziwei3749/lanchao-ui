# Layout 布局 beta

用于布局的容器组件，方便快速搭建页面的基本结构:

  • <l-layout>: 当子元素中包含 <l-sider> 时，全部子元素会水平左右排列，否则会垂直上下排列。
  • <l-header>: 顶栏容器。
  • <l-content>: 主要区域容器。
  • <l-footer>: 底部容器。
  • <l-sider>: 侧边栏容器。

WARNING

以上组件采用了 flex 布局，使用前请确定目标浏览器是否兼容。此外，<l-layout> 的子元素只能是后四者，后四者的父元素也只能是 <l-layout>。
