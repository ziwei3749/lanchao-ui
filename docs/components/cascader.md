# Cascader 级联组件 <Badge text="beta" type="warn"/>

当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择。

## 基础用法

<ClientOnly>
<cascader-demo1 />
</ClientOnly>

            @update:selected="fn"
            popover-height="200px">
</l-cascader>

## 动态加载

当选中某一级时，动态加载该级下的选项

<cascader-demo2 />

```vue
<l-cascader :source.sync="source"
            :selected.sync="selected"
            @update:selected="fn"
            popover-height="200px"
            :load-data="loadData">
</l-cascader>

<script>
    methods(){
        loadData(item, updateSource) {
            let id = item.id;
            ajax(id).then(result => {
                updateSource(result);
            });
        },
    }
</script>
```

## Attributes

|      参数      |      说明      |   类型   | 可选值 | 默认值 |
| :------------: | :------------: | :------: | :----: | :----: |
|     source     |     数据源     |  Array   |   —    |   —    |
|    selected    |  选中项绑定值  |  Array   |   —    |   —    |
| popover-height | 设置弹出层高度 |  String  |   —    | 200px  |
|   load-data    |    异步函数    | Function |   —    |   —    |
